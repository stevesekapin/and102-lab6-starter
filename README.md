Application Name

Campground Explorer

Campground Explorer is an Android application that allows users to browse National Parks and Campgrounds using data from the National Park Service API. The app displays park and campground information such as names, descriptions, and images in a scrollable list. Users can tap on an item to view more detailed information on a separate screen.

Completed Features
Parks Screen

Displays a list of National Parks

Uses RecyclerView to show parks

Displays park name, description, and image

Data retrieved from the National Park Service API

Campgrounds Screen

Displays campground information

Shows campground name and location

Uses RecyclerView with a custom adapter

Bottom Navigation

Users can switch between:

Parks

Campgrounds

Detail Screen

Clicking a park or campground opens a DetailActivity

Displays:

Name

Description

Image

Image Loading

Uses the Glide library to load images from URLs

API Integration

Retrieves data using AsyncHttpClient

Parses JSON responses from the API
